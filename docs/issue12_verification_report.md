## 4. 重大差異レポート

### CRITICAL: 科目No統合の比較条件不足

| 項目 | 内容 |
|------|------|
| 深刻度 | **CRITICAL** |
| 対象 | SM.vb `m科目No統合`, SH.vb `m科目No統合` |
| 修正推奨 | はい |

**Access版（期待動作）:**
```vba
gCmp(科目CD) AND gCmp(科目名)
```
2条件（科目CD + 科目名）で統合判定を行う。

**VB.NET版（現状）:**
```vb
GCmp(KamokuCD)
```
1条件（科目CDのみ）で統合判定を行う。

**影響:**
同一科目CDで異なる科目名を持つレコードが存在する場合、Access版では別レコードとして扱われるが、VB.NET版では同一レコードとして統合されてしまう。仕訳出力結果に差異が生じる可能性がある。

**修正方針:**
VB.NET版の `m科目No統合` メソッドに科目名（KamokuNM）の比較条件を追加する。
