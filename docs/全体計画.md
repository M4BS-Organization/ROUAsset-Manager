# LeaseM4BS マイグレーション 全体計画

## プロジェクト概要

Access版 LeaseM4BS（リース会計管理システム）を VB.NET/WinForms へ移行するプロジェクト。
277画面の移植が完了しており、今回の **4.5日間スプリント（3/13 PM〜3/19）** では
**「システムとして動く最小限」** を目指す。
