## 1. 概要

### 1.1 目的

本ドキュメントは、企業会計基準第34号（ASBJ第34号）に定められたリース会計処理と、LeaseM4BSシステムにおける実装の対応関係を網羅的に整理する。各条文に対する計算ロジック、UIへの反映、実装状況を明確にし、開発・レビュー・監査の参照資料とする。

### 1.2 適用範囲

- LeaseM4BS-1リポジトリの FrmLeaseContractMain（5タブ構成）
- FrmLeaseJudgment（リース判定画面）
- 会計タブ（pgAccounting）の計算ロジック `UpdateAccountingTabValues`

## 3. 計算ロジック詳細

### 3.1 UpdateAccountingTabValues — 計算フロー

FrmLeaseContractMain の会計タブで実行される自動計算の全フロー。

```
入力値の取得
├── 月額賃料 ← pgContract（基本期間セクション）
├── 基本期間月数 ← pgContract
├── 更新予想回数 ← pgContract（延長オプション）
├── 更新月数 ← pgJudgment タブの numExtMonths（延長オプション月数）
├── リース割合 ← pgAccounting入力
├── 非リース割合 ← pgAccounting入力
├── 追加借入利子率 ← pgAccounting入力（例: 0.06）
├── 初期直接費用 ← pgInitial
├── 原状回復費用見積 ← pgInitial
└── リース・インセンティブ ← pgInitial

計算処理
│
├── Step 1: 会計期間の算出（§17）
│   会計期間月数 = 基本期間月数 + 更新予想回数 × 更新月数
│
├── Step 2: 賃料総額
│   賃料総額 = 月額賃料 × 基本期間月数
│
├── Step 3: 算定総額（§22）
│   算定総額 = 月額賃料 × 会計期間月数
│
├── Step 4: リース・非リース配分（§13）
│   リース配分額 = 算定総額 × リース割合 / (リース割合 + 非リース割合)
│
├── Step 5: 現在価値PV（§22 年金現価公式）
│   r = 追加借入利子率 / 12（月次割引率）
│   n = 会計期間月数
│   月額支払 = リース配分額 / n
│   PV = 月額支払 × (1 - (1 + r)^(-n)) / r
│
├── Step 6: 使用権資産（§27）
│   使用権資産 = PV + 初期直接費用 + 原状回復費用 - リース・インセンティブ
│
└── Step 7: リース負債（§22）
    リース負債 = PV

出力先
├── pgAccounting: 会計期間、賃料総額、算定総額、PV
├── pnlResult（未実装）: 使用権資産(2100)、リース負債(1100)
└── 返済スケジュールマトリックス: ROU/Liab/ARO期首〜期末
```

### 3.2 返済スケジュールマトリックス — 期間ごと計算

```
期ごとの計算（i = 1, 2, ... N期）:

■ リース負債 (Liab)
  Liab期首[i] = Liab期末[i-1]  (i=1の場合: PV)
  利息[i] = Liab期首[i] × 月次割引率 × 月数
  元本返済[i] = 月額支払 × 月数 - 利息[i]
  Liab期末[i] = Liab期首[i] - 元本返済[i]

■ 使用権資産 (ROU)
  ROU期首[i] = ROU期末[i-1]  (i=1の場合: 使用権資産当初認識額)
  減価償却[i] = ROU当初認識額 / min(リース期間, 耐用年数) × 月数
  ROU期末[i] = ROU期首[i] - 減価償却[i]

■ 原状回復引当 (ARO)
  ARO期首[i] = ARO期末[i-1]
  ARO増加[i] = ※未実装（見積計算ロジック要定義）
  ARO期末[i] = ARO期首[i] + ARO増加[i]
```

## 5. Q1-Q4 リース判定ロジック

pgJudgment タブに実装済みの判定フロー。

```
Q1: リースの存在
  「契約が、対価と引換えに、特定された資産の使用を支配する権利を
   一定期間にわたり移転するか？」
  → Yes: Q2へ / No: リース非該当

Q2: サプライヤーの実質的代替権
  「サプライヤーが資産を代替する実質的な権利を有するか？」
  → No（代替権なし）: Q3へ / Yes（代替権あり）: リース非該当
  ※ 実装: isLease = q1Yes AndAlso q2No AndAlso q3Yes AndAlso q4Yes
    （rbQ2No.Checked = True のときリース該当として続行）

Q3: 経済的便益
  「資産の使用から生じる経済的便益のほとんどすべてを
   得る権利を有するか？」
  → Yes: Q4へ / No: リース非該当

Q4: 使用の指図
  「資産の使用を指図する権利を有するか？」
  → Yes: リース該当 / No: リース非該当

免除判定:
  短期リース（§41）: リース期間 ≤ 12ヶ月 → 費用処理可
  少額リース（§42）: 原資産の公正価値が少額 → 費用処理可
```
