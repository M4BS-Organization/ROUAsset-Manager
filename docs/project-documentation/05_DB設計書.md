### 1.2 接続設定

**シサンM7**
```xml
<!-- App.config -->
<connectionStrings>
  <add name="M7Database"
       connectionString="Host=localhost;Port=5432;Database=m7_db;Username=xxx;Password=xxx"
       providerName="Npgsql" />
</connectionStrings>
```

**LeaseM4BS**（優先順位）
1. `App.config` → `connectionStrings["LeaseM4BS"]`
2. 環境変数 → `LEASE_M4BS_CONNECTION_STRING`
3. デフォルト → `Host=localhost;Port=5432;Database=lease_m4bs`

> **[05-008] セキュリティ注記**: `DbConnectionManager.vb` のデフォルトフォールバック処理にはユーザー名・パスワードが平文でハードコードされている（git 履歴にも残存）。**本番環境・テスト環境では必ず App.config または環境変数で接続文字列を上書きすること**。デフォルト値はローカル開発専用であり、本番環境でのデフォルト接続は厳禁とする。
