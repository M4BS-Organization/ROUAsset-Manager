> **[04-001] クラス継承について**: 5画面はすべて `Inherits System.Windows.Forms.UserControl` を継承する UserControl として定義されている。FrmFlexMenu の SwitchContent() が UserControl を前提として動的切替を行うため、誤って `System.Windows.Forms.Form` を継承しないよう注意すること。

### 共通仕様

- **クラス継承**: `Inherits System.Windows.Forms.UserControl`（必須）
- **遷移元**: FrmFlexMenu の SwitchContent() で UserControl として動的切替
- **UIパターン**: WinForms Code-Behind（イベント駆動型）
- **命名規則**: 既存コーディング規約に準拠（txt/lbl/btn/cmb/dgv/grp/num/dtp/pnl）
- **データアクセス**: CrudHelper を使用（Using helper As New CrudHelper()）
- **エラーハンドリング**: Try/Catch → MessageBox.Show()
- **参照基準**: 企業会計基準第34号（ASBJ第34号）

> **[04-007] SwitchContent() による入力値リセット仕様**: FrmFlexMenu の SwitchContent() は画面遷移時に既存の UserControl を `_currentContent.Dispose()` で破棄してから新しいインスタンスを生成する。そのため、画面を切り替えると入力途中のデータはすべてリセットされる。入力値の保持が必要な場合は、画面遷移前に DB 保存または呼び出し元への値渡しを実装すること。

#### 利息配分計算（実効金利法）

```

```

| 9 | 繰延税金負債 | Decimal | 120 | 将来加算一時差異 × 法定実効税率 |

#### 差異計算ロジック

```
会計上費用 = 減価償却費(§37) + 支払利息(§22)
税務上費用 = 支払リース料
当期差異 = 会計上費用 - 税務上費用
一時差異累計 = 前期累計 + 当期差異
繰延税金 = 一時差異累計 × 法定実効税率
```
