## 1. はじめに

### 1.1 LeaseM4BSとは

LeaseM4BSは、リース契約の管理および新リース会計基準（企業会計基準第34号/ASBJ第34号）に基づく会計処理を支援するWindowsアプリケーションです。リース契約の登録、リース判定、使用権資産・リース負債の計算を自動化します。

### 1.2 主な機能

- リース契約の登録・管理・検索
- ASBJ第34号に基づくリース判定（Q1-Q4フロー）
- 使用権資産・リース負債の自動計算
- 返済スケジュールの生成
- 使用権資産・月次支払・月次会計・期間残高・税法調整の管理（一部開発中）

## 2. 起動と基本操作

### 2.1 アプリケーション起動

1. デスクトップの「LeaseM4BS」アイコンをダブルクリック
2. メインメニュー画面（FrmFlexMenu）が表示される

### 2.2 メインメニュー画面

起動直後に表示されるポータル画面です。各メニューボタンから機能にアクセスします。

```
┌──────────────────────────────┐
│         LeaseM4BS            │
│                              │
│  [契約書(フレックス)]          │ ← 契約の検索・一覧表示
│  [使用権資産]                 │ ← 使用権資産の一覧（開発中）
│  [月次支払]                   │ ← 月次支払の管理（開発中）
│  [月次会計]                   │ ← 月次仕訳の管理（開発中）
│  [期間残高]                   │ ← 残高推移の表示（開発中）
│  [税法調整]                   │ ← 税務差異の管理（開発中）
│                              │
└──────────────────────────────┘
```

> **注意**: 起動直後はデフォルトで「契約書(フレックス)」画面が表示されます。

## 3. 契約書一覧（FrmFlexContract）

### 3.1 画面概要

登録されたリース契約の一覧を検索・表示する画面です。

### 3.2 操作手順

1. メインメニューの「契約書(フレックス)」ボタンをクリック（起動直後は自動表示）
2. 一覧にリース契約データが表示される
3. 検索条件を入力して「検索」ボタンで絞り込み
4. 契約をダブルクリック（または選択して詳細ボタン）で、リース契約詳細画面を開く

## 4. リース契約詳細（FrmLeaseContractMain）

### 4.1 画面概要

リース契約の詳細情報を入力・管理する中心的な画面です。5つのタブで構成されています。

### 4.2 契約タブ（pgContract）

リース契約の基本情報を入力します。

**入力項目**

| セクション | 項目 | 入力方法 | 必須 |
|---|---|---|---|
| 基本情報 | 契約番号 | 自動採番（LC-年-連番） | ○ |
| 基本情報 | 契約名 | テキスト入力 | ○ |
| 管理情報 | 契約管理部署 | リスト選択 | ○ |
| 管理情報 | 費用負担部署 | リスト選択 | - |
| 資産情報 | 資産一覧 | DataGridViewで入力 | ○ |
| 契約条件 | 更新オプション | リスト選択 | - |
| 契約条件 | 解約オプション | リスト選択 | - |
| 契約条件 | 購入オプション | リスト選択 | - |

**操作のポイント**
- 契約番号は「LC-2026-0001」の形式で自動生成されます
- 資産情報は一覧表で複数資産を登録可能です
- 更新/解約/購入オプションの設定は、会計タブの計算に影響します

### 4.3 初回金タブ（pgInitial）

リース開始時の費用を入力します。これらの値は使用権資産の当初認識額の計算に使用されます。

**初回費用明細（DataGridView: dgvInitialCosts）**

費目ごとの初回費用を明細入力します。

| 列 | 内容 | 備考 |
|---|---|---|
| 費目 | 敷金 / 敷金償却額（返還不能分）/ 礼金 / 仲介手数料 | ドロップダウン選択 |
| 金額（税抜） | 費目ごとの税抜金額 | 手入力 |
| 消費税(10%) | 消費税額（自動計算） | 読取専用（敷金は0%） |
| 金額（税込） | 税込合計（自動計算） | 読取専用 |
| 会計処理 | 資産計上 / 費用処理 / 繰延処理 / 預り金処理 | ドロップダウン選択 |

**その他初回費用（判定・会計連動）**

| 項目 | 説明 | 計算への影響 |
|---|---|---|
| 初期直接費用 | リース契約の交渉・締結に直接起因する費用 | 使用権資産に加算 |
| 原状回復費用見積 | リース終了時の原状回復義務の見積額 | 使用権資産に加算 |
| リース・インセンティブ | 貸主から受領した賃料免除等 | 使用権資産から控除 |

### 4.4 会計タブ（pgAccounting）

> **タブ名称について**: コード上のコメントでは「スケジュールタブ」と呼ばれている箇所がありますが、タブラベルは「会計」（`pgAccounting = New TabPage("会計")`）です。本マニュアルでは「会計タブ」で統一します。

ASBJ第34号に基づく会計計算が自動で実行されます。会計タブは以下の **4セクション**で構成されています。

**表示セクション**

```
┌─＜現契約期間＞ + ＜現支払情報＞ ────────────┐
│  ※リース判定タブ・契約タブの入力値が         │
│    自動転記される（読取専用）                 │
│  ＜現契約期間＞: 契約日/開始日/終了日/契約期間│
│  ＜現支払情報＞: 無償期間/支払回数/月額賃料  │
│                /最初支払日/最終支払日         │
└────────────────────────────────────────────┘
┌─＜会計期間＞ ──────────────────────────────┐
│  更新予想回数 / 開始日 / 会計期間 / 終了日   │
│  リース割合 / 非リース割合 / 維持管理費      │
│  割引率: リース判定タブから自動転記（入力不可）│
│  月額賃料: リース判定タブから自動転記（入力不可）│
│                                              │
│  ┌─ 返済スケジュールマトリックス ──────┐    │
│  │  現在価値 / 使用権資産(ROU)         │    │
│  │  / リース負債(Liab) / ARO          │    │
│  │  各列: 期首/増加/変更増減/減少/期末 │    │
│  └──────────────────────────────────────┘    │
└────────────────────────────────────────────┘
┌─＜変更履歴＞ ──────────────────────────────┐
│  契約変更の履歴が表示される                  │
└────────────────────────────────────────────┘
```

**自動計算の仕組み**

契約タブと初回金タブの入力に基づき、以下が自動計算されます。

1. **会計期間** = 基本期間 + 更新予想回数 × 更新月数
2. **算定総額** = 月額賃料 × 会計期間月数
3. **リース配分額** = 算定総額 × リース割合 / (リース割合 + 非リース割合)
4. **現在価値（PV）** = 月額支払 × (1-(1+r)^-n)/r（年金現価公式）
5. **使用権資産** = PV + 初期直接費用 + 原状回復費用 − リース・インセンティブ
6. **リース負債** = PV

**入力のポイント**
- リース割合と非リース割合は合計100%になるよう入力してください
- **割引率（追加借入利子率）はリース判定タブの「割引率」フィールド（numDiscountRate）で入力します。会計タブでは直接入力できません（自動転記・読取専用）**
- **月額賃料もリース判定タブの「月額賃料」フィールド（numMonthlyRentJudge）から自動転記されます（会計タブでは読取専用）**
- 値を変更すると全タブの関連値が自動再計算されます

### 4.5 転貸タブ（pgSublease）

サブリース契約がある場合の情報を入力します。

**基本情報**

| フィールド名 | 項目 | 説明 |
|---|---|---|
| txtSublesseeName | 転貸先名称 | サブリースの借主名（テキスト入力） |
| txtSubleaseArea | 転貸面積 | 転貸する面積（テキスト入力） |
| dtpSubleaseStart | 転貸開始日 | 転貸期間の開始日（日付選択） |
| dtpSubleaseEnd | 転貸終了日 | 転貸期間の終了日（日付選択） |

**転貸料受取テーブル（DataGridView: dgvSubleaseIncome）**

| 列 | 内容 |
|---|---|
| 科目 | 転貸収入の科目区分 |
| 月額受取額（税抜） | 月額の転貸料（税抜） |
| 消費税 | 消費税額（自動計算） |
| 税込合計 | 税込合計（自動計算） |

### 4.6 リース判定タブ（pgJudgment）

ASBJ第34号に基づくリース判定を行います。4つの質問に順に回答します。

**判定フロー**

```
Q1: 「対価と引換えに、特定された資産の使用を支配する権利を
     一定期間にわたり移転するか？」
     → はい: Q2へ / いいえ: リース非該当

Q2: 「契約の対象が特定された資産であるか？」
     → はい: Q3へ / いいえ: リース非該当

Q3: 「資産の使用から生じる経済的便益のほとんどすべてを
     得る権利を有するか？」
     → はい: Q4へ / いいえ: リース非該当

Q4: 「資産の使用を指図する権利を有するか？」
     → はい: リース該当 / いいえ: リース非該当
```

**免除規定**
- 短期リース（12ヶ月以内）: リース料を発生時に費用処理可能
- 少額リース: 原資産の公正価値が少額の場合、費用処理可能

## 5. テストデータの準備

初めてシステムを使用する際、デモデータで動作確認を行う手順です。

> **重要**: 「システム→データベース管理→復元」メニューは **現在未実装**です。FrmFlexMenu に「システム」メニューや「データベース管理」機能は存在しません。テストデータの準備には以下の代替手順を使用してください。

**代替手順 A: PostgreSQL 直接操作によるデータ投入**

1. pgAdmin 4 またはターミナルで PostgreSQL に接続
2. 対象データベース（`lease_m4bs_dev`）を選択
3. INSERT 文でテストデータを直接登録

**代替手順 B: pg_restore によるダンプファイルからの復元**

```bash
# ダンプファイルからの復元
pg_restore -U postgres -d lease_m4bs_dev /path/to/backup.dump

# または SQL ファイルからの復元
psql -U postgres -d lease_m4bs_dev -f /path/to/testdata.sql
```

> **参考（将来実装予定）**: Access データベース（.mdb ファイル）からの移行機能は開発ロードマップに含まれていますが、現時点では実装されていません。

## 6. よくある質問（FAQ）

**Q: 契約番号を変更したい**
A: 契約番号は自動採番されるため、手動変更はできません。新規登録時に自動生成されます。

**Q: 会計タブの計算結果がおかしい**
A: 以下を確認してください。追加借入利子率は年率で入力されていますか（例: 6%→0.06）。リース割合と非リース割合の合計が0になっていませんか。契約タブの基本情報が正しく入力されていますか。

**Q: 「この機能は現在実装中です」と表示される**
A: ヘッダーの登録・修正・更新・中途解約・削除ボタンは現在開発中です。データの保存機能は今後のアップデートで追加予定です。

**Q: データベースに接続できない**
A: PostgreSQLが起動しているか確認してください。App.configの接続文字列が正しいか確認してください。ネットワーク接続に問題がないか確認してください。

**Q: 使用権資産画面などが空白で表示される**
A: 使用権資産、月次支払、月次会計、期間残高、税法調整の5画面は現在開発中（プレースホルダ状態）です。今後のアップデートで機能が追加されます。

## 8. 用語集

> **注意**: 以下の科目コードは仮の値です。正式な科目コード体系は勘定科目マスタ確定後に更新予定です。

| 用語 | 説明 |
|---|---|
| 使用権資産（ROU） | リース契約に基づき貸借対照表に計上する資産。科目コード2100（※仮） |
| リース負債 | リース支払義務の現在価値。科目コード1100（※仮） |
| PV（現在価値） | 将来のリース料支払を割引率で現在の価値に換算した額 |
| 追加借入利子率 | リース料と同等の支払パターンで借入を行う場合の利子率 |
| 敷金 | 賃貸契約時に預ける保証金。科目コード2210（※仮） |
| ARO（原状回復引当） | リース終了時の原状回復義務に対する引当金 |
| ASBJ第34号 | 企業会計基準第34号「リースに関する会計基準」 |
| フレックスリスト | 動的に列構成を変更可能な一覧表示機能 |
