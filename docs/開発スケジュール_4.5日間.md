## 本スケジュールのスコープ

9人日で完了できる範囲に**最優先タスクのみ**に絞り込む。

### スコープ内（今回実施）

| 優先度 | 項目 | 理由 |
|--------|------|------|
| **最高** | FileHelper.ToFixedLengthFile 完成 | 顧客連携に必須 |
| **最高** | FileHelper.ToCsvFile デリミタ修正 | 仕訳出力の前提 |
| **高** | 一覧画面の計算TODO消化（KEIJO, KHIYO, KLSRYO） | 業務利用に直結 |
| **高** | 標準仕訳出力（KJ/SM/SH）の検証・修正 | コア機能 |
| **高** | セキュリティ：ログイン画面の最低限実装 | 起動フローに必須 |
| **中** | 条件画面のTODO消化（KEIJO_JOKEN, CHUKI_JOKEN等） | 帳票出力の前提 |
| **中** | 仕訳出力設定画面の検証 | 仕訳出力の前提 |

### スコープ外（次フェーズ送り）

| 項目 | 理由 |
|------|------|
| 顧客固有仕訳出力（fc_ 53画面）の個別検証 | 9人日では不可能。次フェーズで顧客別に対応 |
