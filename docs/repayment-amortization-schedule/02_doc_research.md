## Web調査結果: リース会計スケジュール生成ロジック

### 1. IFRS16 / ASC842 における使用権資産の償却方法
- 使用権資産の償却: 定額法が一般的
- リース負債の利息費用: 利息法（実効金利法）
- IFRS16は借手リースを区分しない（全てオンバランス）

### 2. リース負債の返済スケジュール計算方法
- PV = Σ [Payment / (1 + r)^n]
- 実効金利法: 利息額 = 期首残高 × 月利率, 元本返済額 = リース料 − 利息額

### 3. VB.NET での金融計算
- Financial.PV/IPmt/PPmt はDouble型返却 → Decimal変換推奨
- 本プロジェクトはAccess版互換のためDouble型を維持

### 4. 長短振替
- 流動負債: 決算日から12か月以内の元本返済額合計
- 非流動負債: それ以外

### 5. セキュリティ考慮事項
- Decimal vs Double: Access版互換のためDouble維持、GIntで精度保証
